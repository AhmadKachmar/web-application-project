# Full-Stack Web Application

## Overview
This project is a full-stack web application built using HTML, CSS, JavaScript, PHP, and SQL. It includes user authentication, profile handling, feedback features, quiz/progress pages, and leaderboard functionality.

## Features
- User authentication (sign up / sign in / logout)
- Profile and progress pages
- Quiz and practice-related pages
- Leaderboard functionality
- Feedback and contact forms
- Database integration using SQL
- Dynamic interactions using JavaScript, PHP, and AJAX

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- SQL
- AJAX

This project demonstrates full-stack web development concepts including authentication, database integration, server-side scripting, and dynamic data handling.
